x# Technology Stack

## Frontend
- **HTML5:** Provides the semantic structure for the application.
- **CSS3:** Custom styling (Vanilla CSS) to create a premium, modern, and dynamic user interface. The design will utilize rich aesthetics, micro-animations, and responsive layouts.
- **JavaScript (ES6+):** Client-side logic, DOM manipulation, and integration with Firebase (Vanilla JS, no framework).

## Backend & Database (Firebase - Free Spark Plan)
*Selected specifically to meet the criteria of a zero-cost backend. The Firebase Spark plan provides ample free quotas for Authentication, Firestore, and Hosting.*
- **Firebase Authentication:** Handles secure user registration, login, and session management using Email/Password authentication.
- **Cloud Firestore:** A flexible, scalable NoSQL cloud database to store user profiles, 'Lost' reports, 'Found' reports, and item matching data.
- **Firebase Hosting:** Fast and secure hosting for the static web assets (HTML, CSS, JS).

## Development & Operations
- **Version Control:** Git & GitHub.
- **Agentic Coding:** AI Agents will be utilized to generate boilerplate code, assist with testing, optimize performance, and iterate on UI designs.
